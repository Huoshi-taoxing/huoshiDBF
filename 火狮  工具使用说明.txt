- 关于本工具 [目录爆破 DBFT] 的使用 指南:

 >> [目标 URL] 填写目标 网址URL 锁定网址

 >> [字典爆破] 将 TXT 文件字典添加到爆破工具里。

 >> [扩展名] 手动快速写下字典的格式 如 php, com, net, org, html 等

 >> [响应关键词] 输入您想要的获取的 网页 关键词内容或特征，用于在目录爆破过程中检测响应的网页内容。
        如果响应中 包含你指定的     关键词，那就可以判断这个页面“是有意义的”或者“是真实存在的。 

 >> [自定义请求头] 手动填写目标页面的 HTTP 请求头信息，以模拟更真实的浏览器或绕过目标服务器的防护机制。

 >> [精准锁定] 能够完全彻底过滤掉所有无页面的目录网页，精准爆破。

 >> [已发现的目录] 爆破中可以在最上方查看是否爆破出准确的目录信息，点击 后 ctrl + c 复制目录网址即可。
